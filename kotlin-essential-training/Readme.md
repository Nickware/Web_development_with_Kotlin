# Kotlin Essential Training

Kotlin Essential Training: Functions, Collections, and I/O es un curso práctico de LinkedIn Learning diseñado para principiantes que quieren dominar conceptos fundamentales de Kotlin mediante la creación de programas de línea de comandos.

## Detalles del curso
- **Instructor:** Nate Ebel, autor de *Mastering Kotlin*.
- **Duración:** 4 horas y 42 minutos.
- **Nivel:** Principiante.
- **Fecha de lanzamiento:** 29 de abril de 2022.
- **Plataforma:** LinkedIn Learning.[1]

## Temas principales cubiertos
- Fundamentos de Kotlin: tipos de datos, variables, statements, expresiones, funciones y clases.
- **Control de flujo:** `if`, `when`, `try/catch`, bucles `while` y `for`.
- **Colecciones:** manejo eficiente de listas, mapas y conjuntos.
- **Entrada/Salida (I/O):** lectura y escritura de archivos, manejo de consola.
- **Coroutines:** programación asíncrona segura y eficiente.
- **Debugging:** técnicas para depurar código Kotlin.

## Recursos adicionales
- **Repositorio GitHub oficial:** Incluye branches por cada video del curso con código inicial y final de ejercicios.[2]
- **Certificado:** Certificado descargable y compartible en LinkedIn.
- **Quizzes:** 11 evaluaciones para reforzar el aprendizaje.

## Para quién es ideal
Perfecto para desarrolladores que:
- Quieren aprender Kotlin desde cero.
- Buscan crear aplicaciones de línea de comandos.
- Necesitan entender funciones, colecciones y operaciones I/O.
- Planean migrar desde Java o explorar desarrollo Android/backend.

Este curso proporciona una base sólida para luego avanzar a desarrollo móvil, web o ciencia de datos con Kotlin, temas que hemos discutido previamente en nuestra conversación.
